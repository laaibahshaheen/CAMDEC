# CAMDEC - Camera Detection System

## Overview

CAMDEC is a real-time object detection system built using **Python, OpenCV, and YOLOv8**. The application captures live video from a webcam and performs object detection in real time, displaying detected objects along with confidence scores and performance metrics.

This project was developed to explore modern computer vision techniques and gain hands-on experience with real-time AI inference pipelines.

---

## Features

* Real-time object detection using webcam feed
* Live object recognition using YOLOv8
* Bounding box visualization
* Confidence score display
* Frames Per Second (FPS) monitoring
* Lightweight and easy to run

---

```bash

```

```bash
pip install -r requirements.txt
```
